# Network Packet Analyzer

## Overview

The Network Packet Analyzer is a Python-based application designed to capture and analyze network traffic in real time. It provides detailed information about packets, including source IP, destination IP, protocol type, packet length, and payload data.

## Features

* Real-time packet capture and monitoring
* Displays source and destination IP addresses
* Detects packet protocols (TCP, UDP, ICMP, etc.)
* Shows packet length and payload information
* Clear packet logs
* Export captured data to CSV format
* User-friendly graphical interface

## Technologies Used

* Python
* Tkinter (GUI)
* Scapy (Packet Capture)
* Pandas (CSV Export)
